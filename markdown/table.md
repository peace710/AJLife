# Markdown 表格
Markdown 制作表格使用 `| `来分隔不同的单元格,使用` - `来分隔表头和其他行
```
|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
```
|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

#### 对齐方式
设置表格的对齐方式  
`-:`  设置内容和标题栏居右对齐  
`:-`  设置内容和标题栏居左对齐  
`:-:` 设置内容和标题栏居中对齐

```
| 左对齐(Left) | 右对齐(Right) | 居中对齐(Center) |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

| 左对齐(Left) | 右对齐(Right) | 居中对齐(Center) |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
